scatter_salary_age.png：年齢と収入の散歩図を見ると、この社会では30歳前後の時に給料が一番大きく上がっていることが分かりました。
25歳から30歳までの若い社員は、給料の差があまり大きくありません。
また、40歳から45歳くらいの社員は収入が高いです。全体的に見ると、年齢が上がるにつれて給料も高くなる傾向があります。

Average Salary by Department.png：このグラフでは、部門ごとの平均給与を比較しています。
Finance 部門の平均給与が最も高い。
HR 部門と Marketing 部門は、比較的給与水準が低い。
給与差は、職種の専門性や求められる経験の違いを反映している可能性がある。
